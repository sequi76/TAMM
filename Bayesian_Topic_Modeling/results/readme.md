## Folder with results

The content of this folder are mailny `.pickle` files that are produced with the noteboks in the parent directory.

Because of files size (about 500M) we have gzipped and stored in Zenodo the files in this folder.  To complete this folder, once this repo is cloned in your local, download here the file <a href='https://zenodo.org/records/19341120/files/results.tar.gz?download=1'>results.tar.gz</a>, untar it with `tar -xzf results.tar.gz`, and then you can run the python notebook `plots-paper.ipynb` from the parent directory to create the paper plots.
