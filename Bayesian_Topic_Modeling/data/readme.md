# Data folder

This directory whould contain 2 files and 2 folders:

- `dihiggs_sd_processed.dat` (file)
- `bbbb_sd_combined_processed_feb_12.dat` (file)
- `signal_ssds_big-var_processed` (folder)
- `background_ssds_big-var_processed` (folder)

Everything should be dowloaded from <a href="https://zenodo.org/records/19341120">zenodo.org/records/19341120</a>.  The files are directly downloaded with the same name they have in the Zenodo repo.  The folders instead one should download and untar the files `background_ssds_processed_big_var.tar.gz` and `signal_ssds_processed_big_var.tar.gz`.  When untarring, each file will have a folder named `processed`, one should pour the files inside `processed` into the above folders that should be created in your local.
