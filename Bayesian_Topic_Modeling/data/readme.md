# Data folder

This directory should contain 2 files and 2 folders into your local:

- `dihiggs_sd_processed.dat` (file to be downloaded from Zenodo)
- `bbbb_sd_combined_processed_feb_12.dat` (file to be downloaded from Zenodo)
- `signal_ssds_big-var_processed` (folder currently empty, fill it from Zenodo following below instructions)
- `background_ssds_big-var_processed` (folder currently empty, fill it from Zenodo following below instructions)

Because of file sizes, everything should be dowloaded from <a href="https://zenodo.org/records/19341120">zenodo.org/records/19341120</a>.  The files are directly downloaded with the same name they have in the Zenodo repo.  The folders instead one should download and untar the files `background_ssds_processed_big_var.tar.gz` and `signal_ssds_processed_big_var.tar.gz`.  When untarring, each file will have a folder named `processed`, one should pour the files inside `processed` into the above folders that are currently empty.  Put signal in signal and background in background.
